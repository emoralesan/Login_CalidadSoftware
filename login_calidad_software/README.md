# Proyecto Login - Calidad de Software

Aplicación front-end desarrollada con **Astro**, **TailwindCSS**, **TypeScript** y una arquitectura modular y profesional.  

Incluye sistema de autenticación con **Axios**, manejo de tokens, validación de formularios, **Sonner** para toasts, **loader global** basado en cssloaders.github.io y middleware para proteger rutas.


nota: hay console.log y debugger puestos para facilitar la revisión del código.
